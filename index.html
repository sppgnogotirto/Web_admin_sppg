Pengaturan API

Hubungkan dashboard dengan Google Sheets melalui Google Apps Script.

Deploy Apps Script sebagai Web App, pilih akses "Anyone".
  1. Sheet dibuat otomatis oleh aplikasi Android dengan format: PERSIAPAN_yyyy-MM-dd, PERSIAPAN_LIMBAH_yyyy-MM-dd, PENGOLAHAN_yyyy-MM-dd, PENGOLAHAN_PRODUKSI_yyyy-MM-dd, PEMORSIAN_OMPRENG_yyyy-MM-dd, PEMORSIAN_SISA_yyyy-MM-dd, DISTRIBUSI_yyyy-MM-dd, PENCUCIAN_LIMBAH_yyyy-MM-dd, dan KEBERSIHAN_LIMBAH_yyyy-MM-dd.
  2. Buka Extensions > Apps Script.
  3. Salin kode API website ke editor Apps Script.
  4. Klik Deploy > New deployment, lalu pilih Web App.
  5. Atur Execute as: Me dan Who has access: Anyone.
  6. Salin URL Web App dan tempel di kolom di atas.
